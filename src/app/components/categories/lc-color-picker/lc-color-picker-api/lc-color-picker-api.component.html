API del Componente Color Picker

El componente Color Picker proporciona una variedad de propiedades y métodos que permiten a los desarrolladores personalizar y controlar su comportamiento, apariencia y funcionalidad. A continuación, se presentan las propiedades y métodos más importantes disponibles en el componente Color Picker:

Propiedades

color: string

La propiedad `color` almacena el color actual seleccionado en el Color Picker. Puede ser representado en formatos como hexadecimal, RGB o HSL. Cambiar esta propiedad actualizará la selección visual del color en el componente y reflejará el color seleccionado por el usuario.

paleta: string[]

La propiedad `paleta` define la paleta de colores disponibles para que los usuarios elijan. Puede ser un arreglo de colores en formato hexadecimal. Los colores de esta paleta serán presentados en el Color Picker para que los usuarios puedan seleccionarlos.

Métodos

seleccionarColor(color: string): void

Este método permite a los desarrolladores seleccionar un color específico en el Color Picker. El parámetro `color` debe ser un valor de color válido en formato hexadecimal, RGB o HSL. Llamar a este método cambiará la selección actual del color en el componente.

restablecerColor(): void

El método `restablecerColor` restablece el color seleccionado al valor inicial. Esto es útil cuando deseas permitir a los usuarios volver al color predeterminado o eliminar la selección actual de color.

Eventos

(colorCambiado): EventEmitter<string>

Este evento se emite cuando el color seleccionado en el Color Picker cambia. Los suscriptores pueden escuchar este evento para realizar acciones adicionales en respuesta al cambio de color. El evento emite el nuevo color seleccionado como un valor de cadena.

Personalización y Integración

El componente Color Picker permite personalizar la paleta de colores, los formatos admitidos y el diseño visual del selector de colores. Puedes integrar el Color Picker con otros componentes, como campos de entrada, para crear una experiencia de selección de color cohesiva en toda tu aplicación.

En resumen, la API del componente Color Picker proporciona una serie de propiedades y métodos que permiten a los desarrolladores controlar y personalizar la selección de colores en su aplicación. Con estas herramientas, puedes crear una experiencia de usuario única y atractiva en relación con la elección de colores personalizados.
