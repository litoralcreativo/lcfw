Ejemplos del Componente Tabla

En esta sección, se presentan algunos ejemplos prácticos de cómo utilizar el componente de tabla en diferentes escenarios. Estos ejemplos ayudarán a los desarrolladores a comprender cómo implementar y personalizar el componente para satisfacer sus necesidades específicas.

Ejemplo 1: Tabla Básica

En este ejemplo, mostraremos cómo crear una tabla simple con datos predefinidos. La tabla contendrá columnas como "Nombre", "Edad" y "País". 

<app-tabla [data]="datos" [columnas]="columnas" [orden]="ordenInicial"></app-tabla>

En este caso, `datos` es un arreglo de objetos que contiene información sobre las personas, `columnas` define las columnas que se mostrarán en la tabla, y `ordenInicial` establece la columna inicial de ordenamiento.

Ejemplo 2: Tabla con Filtros

En este ejemplo, mostraremos cómo agregar la capacidad de filtrar los datos en la tabla. Los usuarios podrán ingresar un valor en el campo de filtro para mostrar solo los datos que coincidan. 

<app-tabla [data]="datos" [columnas]="columnas" [orden]="ordenInicial"></app-tabla>

Además de la tabla básica, hemos agregado un campo de entrada para el filtro. El método `filtrar(dato)` se llama cuando los usuarios ingresan un valor en el campo. Esto permite actualizar la tabla y mostrar solo los datos relevantes.

Ejemplo 3: Tabla con Paginación

En este ejemplo, mostraremos cómo implementar la paginación en la tabla para dividir los datos en páginas más pequeñas. Esto es útil cuando se tienen grandes conjuntos de datos. 

<app-tabla [data]="datos" [columnas]="columnas" [orden]="ordenInicial"></app-tabla>

Hemos agregado una funcionalidad de paginación que permite a los usuarios navegar a través de las páginas de datos. Cada página muestra un número limitado de entradas, lo que mejora la experiencia del usuario.

Estos ejemplos proporcionan una visión general de cómo se puede utilizar y personalizar el componente de tabla en diferentes situaciones. Los desarrolladores pueden aprovechar estas ideas como punto de partida para crear interfaces de usuario eficientes y atractivas que muestren y gestionen datos de manera efectiva.
